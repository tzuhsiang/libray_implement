# LibriFlow Manager 使用範例

這裡提供一些常見的使用場景範例。

## 範例 1：查詢與狀態更新
**使用者：** 「幫我看看我有什麼還沒讀的書？」
**助理：** (呼叫 `list_books`)
**助理：** 「您目前有以下未讀書籍：
1. 《三體》- 劉慈欣
2. 《基地》- 艾西莫夫」
**使用者：** 「把《三體》設為正在閱讀。」
**助理：** (呼叫 `update_book_status`)
**助理：** 「好的，已將《三體》的狀態更新為『閱讀中』。」

## 範例 2：新增與評分
**使用者：** 「幫我新增一本新書，書名是《子彈筆記》，作者是 Ryder Carroll，狀態設為已讀，評分 5 分。」
**助理：** (呼叫 `add_book`)
**助理：** 「成功新增書籍《子彈筆記》（ID: 15），並已設定為『已讀』且評分 5 分。」
